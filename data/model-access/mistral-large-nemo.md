## Summary

Mistral's open model family ranges from consumer-friendly Nemo to much larger self-hosted models approaching frontier quality.

## Deployment Modes

- self-hosted
- local-on-device
- local-api

## Common Runtimes

- Ollama
- LM Studio

## Constraints

- runtime practicality depends heavily on which model size is chosen
- larger variants need serious GPU capacity
- the model family should not be treated as a single hardware profile
