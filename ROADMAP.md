# Roadmap

Planned features and enhancements for AI Feature Tracker.

## High Priority

### Search / Quick Filter
Add a text search box to filter features by name or keyword. Useful as the feature count grows.

### "What's New" Badge
Show a visual indicator (e.g., "NEW" badge) on features updated within the last 30 days. Uses the existing `Verified` date field.

### Export Options
- **CSV export** — Download filtered results as CSV for spreadsheets
- **JSON export** — Machine-readable format for integrations

### Comparison View
Side-by-side comparison of 2-3 selected features or platforms. Helps users evaluate alternatives.

---

## Medium Priority

### Changelog Modal Improvements
- Show full changelog history when clicking the Launched date
- Add visual timeline of feature updates

### Mobile Responsiveness Audit
Review and improve the mobile experience:
- Filter controls on small screens
- Card layout optimization
- Touch target verification

### RSS/Atom Feed
Auto-generated feed of feature updates for users who want notifications.

### Feature Request System
- GitHub issue template for requesting new feature entries
- Voting or reactions on feature requests

---

## Low Priority / Future Ideas

### API Endpoint
Static JSON API endpoint for programmatic access to feature data.

### Browser Extension
Quick lookup of AI features from any page.

### Historical Snapshots
Archive previous states of feature availability (e.g., "What was available 6 months ago?").

### i18n Support
Internationalization for non-English audiences.

### Analytics Dashboard
Aggregate statistics:
- Features per platform
- Free vs paid feature ratios
- Category breakdowns
