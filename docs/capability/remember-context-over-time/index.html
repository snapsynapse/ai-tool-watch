    Remember Context Over Time

    Can retain preferences, memory, or persistent context across sessions.

    Definition

    What counts

    • explicit memory features
    • saved preferences reused later
    • persistent user context across chats

    What doesn't count

    • only remembering content inside a single active session
    • temporary context windows with no persistence

    Product Availability (4 products)

    ChatGPT

    Memory

    Memory lets ChatGPT remember your preferences and context across conversations. **All plans get saved memories**; Plus/Pro also get full chat history reference for richer personalization.

    Memory

    Gemini

    Past Chats (Memory)

    Gemini's Past Chats feature lets it remember your preferences and context across conversations. Enable it in Settings → Personal context. **Now free for all users**—it rolled out to non-paying accounts in late February 2026. You can control what Gemini remembers via 'Instructions for Gemini' in settings.

    Grok

    Memory

    Grok has persistent memory that carries user preferences and context across conversations. **Available on all tiers including free.** Memory is selective and automatic—Grok stores key facts it deems important, but users cannot currently inspect or edit what has been saved. The feature appeared with Grok 3 in early 2025 and behavior can be inconsistent across platforms.
